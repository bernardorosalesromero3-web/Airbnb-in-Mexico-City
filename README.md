
# Airbnb CDMX 2023 — Análisis Exploratorio

Análisis de los listados de Airbnb en la Ciudad de México para 2023,
con foco en distribución geográfica, estructura de precios e indicadores
de uso comercial vs. personal como proxy del impacto en el mercado de vivienda.

## Dataset

Fuente: [Airbnb Mexico City 2023 — Kaggle](https://www.kaggle.com/datasets/hirampaulin/airbnb-mexico-city-2023)

Los datos no están incluidos en este repositorio. Descarga el archivo CSV
desde Kaggle y colócalo en una carpeta `data/` en la raíz del proyecto.
