# Airbnb-in-Mexico-City
Análisis de datos de Airbnb CDMX (2023) enfocado en la distribución geográfica y estructura de precios. El proyecto evalúa la proliferación de anfitriones comerciales y la disponibilidad inmobiliaria, aportando métricas clave al debate actual sobre la gentrificación, el uso del suelo y la presión en el mercado de vivienda local.
